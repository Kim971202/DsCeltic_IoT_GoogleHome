Service Overview


The Daesung Integrated IoT Service allows you to efficiently manage and control boilers installed in your home.


▶ Supported Products
  - DR-910W
  - DCR-91WF

▶ Control Devices/Services
  After signing up for the Daesung Smart Home, you can control connected devices and check service information.

[Home Control]
  - Operation Mode: Control the modes supported by each boiler.
  - Water Temperature: Control the water temperature supported by each boiler.
  - Heating Temperature: Control the heating temperature supported by each boiler.
  - Reservation Time: Set the time for reservation mode supported by each boiler.

[User Features]
  - User Group: Add users to your group to share devices.
  - Location Change: Change the location information of your installed device.
  - Away/Home: Control boilers based on your distance.
  - Monitoring: Check boiler usage time.

[Settings]
  - Member Information Settings: Check registered member and household member information.
  - App Version: Check and update the app version.
  - Push Notification Settings: Set up push notifications.

▣ Supported OS Versions for Daesung Smart Home App
  - Android 7 (NOUGAT) or higher
  - iOS 13 or higher

※ Supported features may vary depending on site options and installed products.
